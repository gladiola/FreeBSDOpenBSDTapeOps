# FreeBSDOpenBSDTapeOps (Français)

Scripts shell interactifs pour les opérations courantes sur bande magnétique avec mt et tar.

## Utilisation rapide

- Exécutez `./scriptedDemo.sh` sur FreeBSD ou `./scriptedDemo_openbsd.sh` sur OpenBSD.
- Confirmez que la bande est chargée puis suivez chaque invite avec Entrée.
- Utilisez les scripts de `scripts/` pour l’archivage A→B→C, le transfert, l’écriture sur bande, l’inventaire et la restauration.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
