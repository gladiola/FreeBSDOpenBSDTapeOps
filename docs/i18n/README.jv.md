# FreeBSDOpenBSDTapeOps (Basa Jawa)

Skrip shell interaktif kanggo operasi pita magnet sing umum nganggo mt lan tar.

## Panggunaan cepet

- Lakokna `./scriptedDemo.sh` ing FreeBSD utawa `./scriptedDemo_openbsd.sh` ing OpenBSD.
- Priksa manawa pita wis dimuat lan pencet Enter ing saben langkah.
- Gunakna skrip ing `scripts/` kanggo arsip A→B→C, transfer, nulis pita, inventaris, lan mulihake.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
