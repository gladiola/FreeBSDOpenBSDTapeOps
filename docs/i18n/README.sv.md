# FreeBSDOpenBSDTapeOps (Svenska)

Interaktiva shellskript för vanliga magnetbandsoperationer med mt och tar.

## Snabbstart

- Kör `./scriptedDemo.sh` på FreeBSD eller `./scriptedDemo_openbsd.sh` på OpenBSD.
- Bekräfta att bandet är laddat och tryck Enter vid varje steg.
- Använd skripten i `scripts/` för A→B→C-arkivering, överföring, bandskrivning, inventering och återställning.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
