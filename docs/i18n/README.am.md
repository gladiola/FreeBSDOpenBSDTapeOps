# FreeBSDOpenBSDTapeOps (አማርኛ)

በ mt እና tar የሚሰሩ የመግነጢሳዊ ቴፕ መደበኛ ስራዎች የተሳትፎ ሼል ስክሪፕቶች።

## ፈጣን አጠቃቀም

- በ FreeBSD `./scriptedDemo.sh` ወይም በ OpenBSD `./scriptedDemo_openbsd.sh` ያስኪዱ።
- ቴፕ እንደተጫነ ያረጋግጡ እና በእያንዳንዱ ደረጃ Enter ይጫኑ።
- ለ A→B→C ማህደር፣ ማስተላለፍ፣ ወደ ቴፕ መጻፍ፣ ኢንቨንተሪ እና መመለስ `scripts/` ውስጥ ያሉ ስክሪፕቶችን ይጠቀሙ።

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
