# FreeBSDOpenBSDTapeOps (Deutsch)

Interaktive Shell-Skripte für häufige Magnetband-Operationen mit mt und tar.

## Kurzanleitung

- Führen Sie unter FreeBSD `./scriptedDemo.sh` oder unter OpenBSD `./scriptedDemo_openbsd.sh` aus.
- Bestätigen Sie das eingelegte Band und folgen Sie den Eingabeaufforderungen mit Enter.
- Nutzen Sie die Skripte in `scripts/` für A→B→C-Archivierung, Übertragung, Bandschreiben, Inventur und Wiederherstellung.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
