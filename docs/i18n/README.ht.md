# FreeBSDOpenBSDTapeOps (Kreyòl Ayisyen)

Script shell entèaktif pou operasyon kasèt mayetik komen ak mt ak tar.

## Itilizasyon rapid

- Kouri `./scriptedDemo.sh` sou FreeBSD oswa `./scriptedDemo_openbsd.sh` sou OpenBSD.
- Konfime kasèt la chaje epi suiv chak etap ak Enter.
- Sèvi ak script nan `scripts/` pou achiv A→B→C, transfè, ekriti sou kasèt, envantè, ak restorasyon.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
