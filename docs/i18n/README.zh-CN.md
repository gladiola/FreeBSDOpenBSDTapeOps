# FreeBSDOpenBSDTapeOps (简体中文)

使用 mt 和 tar 的交互式 shell 脚本，用于常见磁带操作。

## 快速使用

- 在 FreeBSD 运行 `./scriptedDemo.sh`，或在 OpenBSD 运行 `./scriptedDemo_openbsd.sh`。
- 确认磁带已装入，并在每一步按 Enter 继续。
- 使用 `scripts/` 中的脚本完成 A→B→C 归档、传输、写带、盘点和恢复。

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
