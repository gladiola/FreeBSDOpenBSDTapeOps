# FreeBSDOpenBSDTapeOps (Bahasa Melayu)

Skrip shell interaktif untuk operasi pita magnet biasa menggunakan mt dan tar.

## Penggunaan pantas

- Jalankan `./scriptedDemo.sh` di FreeBSD atau `./scriptedDemo_openbsd.sh` di OpenBSD.
- Sahkan pita telah dimuatkan dan tekan Enter pada setiap langkah.
- Gunakan skrip dalam `scripts/` untuk pengarkiban A→B→C, pemindahan, penulisan pita, inventori, dan pemulihan.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
