# FreeBSDOpenBSDTapeOps (Tagalog)

Mga interaktibong shell script para sa karaniwang operasyon ng magnetic tape gamit ang mt at tar.

## Mabilis na paggamit

- Patakbuhin ang `./scriptedDemo.sh` sa FreeBSD o `./scriptedDemo_openbsd.sh` sa OpenBSD.
- Kumpirmahing naka-load ang tape at pindutin ang Enter sa bawat hakbang.
- Gamitin ang mga script sa `scripts/` para sa A→B→C na pag-archive, paglipat, pagsusulat sa tape, imbentaryo, at pag-restore.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
