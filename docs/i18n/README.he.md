# FreeBSDOpenBSDTapeOps (עברית)

סקריפטי shell אינטראקטיביים לפעולות סרט מגנטי נפוצות באמצעות mt ו‑tar.

## שימוש מהיר

- הפעילו `./scriptedDemo.sh` ב‑FreeBSD או `./scriptedDemo_openbsd.sh` ב‑OpenBSD.
- אשרו שהסרט נטען ולחצו Enter בכל שלב.
- השתמשו בסקריפטים שבתיקיית `scripts/` לארכוב A→B→C, העברה, כתיבה לסרט, מיפוי ושחזור.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
