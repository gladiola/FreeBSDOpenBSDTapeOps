# FreeBSDOpenBSDTapeOps (Gagana Samoa)

Skripi shell fegalegalea Ľi mo galuega masani o le lipine maneta e fa Ľaaoga ai mt ma tar.

## Fa Ľaoga vave

- Tamo Ľe `./scriptedDemo.sh` i FreeBSD po Ľo `./scriptedDemo_openbsd.sh` i OpenBSD.
- Fa Ľamaonia ua utaina le lipine ma mulimuli i la Ľasaga ta ľitasi i le Enter.
- Fa Ľaoga skripi i `scripts/` mo le teuina A‚ÜíB‚ÜíC, lafoina, tusitusi i le lipine, lisi, ma toe fa Ľaleleia.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
