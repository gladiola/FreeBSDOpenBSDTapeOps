# FreeBSDOpenBSDTapeOps (Bahasa Indonesia)

Skrip shell interaktif untuk operasi pita magnetik umum menggunakan mt dan tar.

## Penggunaan cepat

- Jalankan `./scriptedDemo.sh` di FreeBSD atau `./scriptedDemo_openbsd.sh` di OpenBSD.
- Pastikan pita sudah dimuat lalu ikuti setiap langkah dengan menekan Enter.
- Gunakan skrip di `scripts/` untuk pengarsipan A→B→C, transfer, penulisan ke pita, inventaris, dan pemulihan.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
