# FreeBSDOpenBSDTapeOps (Norsk)

Interaktive shell-skript for vanlige magnetbåndoperasjoner med mt og tar.

## Hurtig bruk

- Kjør `./scriptedDemo.sh` på FreeBSD eller `./scriptedDemo_openbsd.sh` på OpenBSD.
- Bekreft at båndet er lastet og trykk Enter ved hvert trinn.
- Bruk skriptene i `scripts/` for A→B→C-arkivering, overføring, båndskriving, inventar og gjenoppretting.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
