# FreeBSDOpenBSDTapeOps (Te Reo Māori)

Ngā hōtuhi shell pāhekoheko mō ngā mahi rīpene autō noa mā te mt me te tar.

## Whakamahi tere

- Whakahaerehia `./scriptedDemo.sh` i FreeBSD, `./scriptedDemo_openbsd.sh` rānei i OpenBSD.
- Whakaū kua utaina te rīpene, ā, pēhia te Enter i ia taahiraa.
- Whakamahia ngā hōtuhi i `scripts/` mō te pūranga A→B→C, te whakawhiti, te tuhi rīpene, te rārangi, me te whakaora.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
