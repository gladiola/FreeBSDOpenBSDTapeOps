# FreeBSDOpenBSDTapeOps (US English)

Interactive shell scripts for common magnetic tape operations using mt and tar.

## Quick Usage

- Run `./scriptedDemo.sh` on FreeBSD or `./scriptedDemo_openbsd.sh` on OpenBSD.
- Confirm tape loading, then follow each prompt and press Enter to continue.
- Use scripts in `scripts/` for A→B→C archiving, transfer, tape writing, inventory, and restore.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
