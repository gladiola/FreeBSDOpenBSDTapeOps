# FreeBSDOpenBSDTapeOps (繁體中文（香港）)

使用 mt 與 tar 的互動式 shell 腳本，用於常見磁帶操作。

## 快速使用

- 在 FreeBSD 執行 `./scriptedDemo.sh`，或在 OpenBSD 執行 `./scriptedDemo_openbsd.sh`。
- 確認磁帶已載入，依照每步提示按 Enter 繼續。
- 使用 `scripts/` 內腳本完成 A→B→C 封存、傳輸、寫帶、盤點與還原。

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
