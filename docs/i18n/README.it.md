# FreeBSDOpenBSDTapeOps (Italiano)

Script shell interattivi per operazioni comuni su nastro magnetico con mt e tar.

## Uso rapido

- Esegui `./scriptedDemo.sh` su FreeBSD o `./scriptedDemo_openbsd.sh` su OpenBSD.
- Conferma che il nastro è caricato e segui i prompt premendo Invio.
- Usa gli script in `scripts/` per archiviazione A→B→C, trasferimento, scrittura su nastro, inventario e ripristino.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
