# FreeBSDOpenBSDTapeOps (Kiswahili)

Hati za shell za maingiliano kwa shughuli za kawaida za tepu ya sumaku kwa kutumia mt na tar.

## Matumizi ya haraka

- Endesha `./scriptedDemo.sh` kwenye FreeBSD au `./scriptedDemo_openbsd.sh` kwenye OpenBSD.
- Thibitisha tepu imeingizwa kisha fuata kila hatua kwa kubonyeza Enter.
- Tumia hati za `scripts/` kwa uhifadhi wa A→B→C, uhamisho, uandishi wa tepu, orodha na urejeshaji.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
