# FreeBSDOpenBSDTapeOps (العربية)

نصوص شِل تفاعلية لعمليات الشريط المغناطيسي الشائعة باستخدام mt و tar.

## الاستخدام السريع

- شغّل `./scriptedDemo.sh` على FreeBSD أو `./scriptedDemo_openbsd.sh` على OpenBSD.
- أكد تحميل الشريط ثم اتبع كل خطوة واضغط Enter للمتابعة.
- استخدم سكربتات `scripts/` لأرشفة A→B→C والنقل والكتابة على الشريط والجرد والاستعادة.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
