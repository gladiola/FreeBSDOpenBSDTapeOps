# FreeBSDOpenBSDTapeOps (বাংলা)

mt এবং tar ব্যবহার করে সাধারণ ম্যাগনেটিক টেপ অপারেশনের জন্য ইন্টারঅ্যাক্টিভ শেল স্ক্রিপ্ট।

## দ্রুত ব্যবহার

- FreeBSD-তে `./scriptedDemo.sh` বা OpenBSD-তে `./scriptedDemo_openbsd.sh` চালান।
- টেপ লোড হয়েছে নিশ্চিত করে প্রতিটি ধাপে Enter চাপুন।
- `scripts/` এর স্ক্রিপ্ট দিয়ে A→B→C আর্কাইভ, ট্রান্সফার, টেপে লেখা, ইনভেন্টরি ও রিস্টোর করুন।

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
