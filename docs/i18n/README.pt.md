# FreeBSDOpenBSDTapeOps (Português)

Scripts de shell interativos para operações comuns de fita magnética com mt e tar.

## Uso rápido

- Execute `./scriptedDemo.sh` no FreeBSD ou `./scriptedDemo_openbsd.sh` no OpenBSD.
- Confirme que a fita está carregada e siga cada passo pressionando Enter.
- Use os scripts em `scripts/` para arquivamento A→B→C, transferência, gravação em fita, inventário e restauração.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
