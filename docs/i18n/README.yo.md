# FreeBSDOpenBSDTapeOps (Yorùbá)

Àwọn àkọsílẹ shell aláṣiṣẹpọ fún iṣẹ́ teepu amóhùnmáwòrán tí ó wọ́pọ̀ pẹ̀lú mt àti tar.

## Lílò kíákíá

- Sá `./scriptedDemo.sh` lórí FreeBSD tàbí `./scriptedDemo_openbsd.sh` lórí OpenBSD.
- Jẹ́rìí pé teepu ti wọlé, lẹ́yìn náà tẹ Enter ní gbogbo ìgbésẹ̀.
- Lo àwọn àkọsílẹ nínú `scripts/` fún ìpamọ́ A→B→C, ìfiránṣẹ́, kọ sí teepu, àkójọ, àti ìmúpadàbọ̀sípò.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
