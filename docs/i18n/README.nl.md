# FreeBSDOpenBSDTapeOps (Nederlands)

Interactieve shellscripts voor veelvoorkomende magneetbandbewerkingen met mt en tar.

## Snel gebruik

- Voer `./scriptedDemo.sh` uit op FreeBSD of `./scriptedDemo_openbsd.sh` op OpenBSD.
- Bevestig dat de tape is geladen en druk bij elke stap op Enter.
- Gebruik de scripts in `scripts/` voor A→B→C-archivering, overdracht, tape-schrijven, inventarisatie en herstel.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
