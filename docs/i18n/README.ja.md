# FreeBSDOpenBSDTapeOps (日本語)

mt と tar を使った一般的な磁気テープ操作のための対話型シェルスクリプトです。

## クイックスタート

- FreeBSD では `./scriptedDemo.sh`、OpenBSD では `./scriptedDemo_openbsd.sh` を実行します。
- テープ装填を確認し、各手順で Enter を押して進みます。
- `scripts/` のスクリプトで A→B→C のアーカイブ、転送、テープ書き込み、目録作成、復元を行います。

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
