# FreeBSDOpenBSDTapeOps (Suomi)

Interaktiiviset shell-skriptit yleisiin magneettinauhaoperaatioihin mt- ja tar-työkaluilla.

## Pikaohje

- Aja FreeBSD:ssä `./scriptedDemo.sh` tai OpenBSD:ssä `./scriptedDemo_openbsd.sh`.
- Vahvista, että nauha on ladattu, ja paina Enter jokaisessa vaiheessa.
- Käytä `scripts/`-hakemiston skriptejä A→B→C-arkistointiin, siirtoon, nauhakirjoitukseen, inventointiin ja palautukseen.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
