# FreeBSDOpenBSDTapeOps (Eesti)

Interaktiivsed shelliskriptid tavaliste magnetlindi toimingute jaoks, kasutades mt ja tar.

## Kiirkasutus

- Käivita FreeBSD-s `./scriptedDemo.sh` või OpenBSD-s `./scriptedDemo_openbsd.sh`.
- Kinnita, et lint on laaditud, ja vajuta igal sammul Enter.
- Kasuta `scripts/` skripte A→B→C arhiveerimiseks, edastuseks, lindile kirjutamiseks, inventuuriks ja taastamiseks.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
