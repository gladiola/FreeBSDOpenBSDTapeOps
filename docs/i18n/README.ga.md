# FreeBSDOpenBSDTapeOps (Gaeilge)

Scripteanna blaosc idirghníomhacha do ghnáth-oibríochtaí téipe maighnéadaí le mt agus tar.

## Úsáid thapa

- Rith `./scriptedDemo.sh` ar FreeBSD nó `./scriptedDemo_openbsd.sh` ar OpenBSD.
- Deimhnigh go bhfuil an téip luchtaithe agus brúigh Enter ag gach céim.
- Úsáid na scripteanna i `scripts/` le haghaidh cartlannú A→B→C, aistriú, scríobh ar théip, fardal agus aisghabháil.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
