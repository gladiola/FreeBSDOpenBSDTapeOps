# FreeBSDOpenBSDTapeOps (हिन्दी)

mt और tar का उपयोग करके सामान्य मैग्नेटिक टेप कार्यों के लिए इंटरैक्टिव शेल स्क्रिप्ट।

## त्वरित उपयोग

- FreeBSD पर `./scriptedDemo.sh` या OpenBSD पर `./scriptedDemo_openbsd.sh` चलाएँ।
- टेप लोड होने की पुष्टि करें और हर चरण पर Enter दबाएँ।
- `scripts/` की स्क्रिप्ट्स से A→B→C आर्काइव, ट्रांसफर, टेप लेखन, इन्वेंटरी और रिस्टोर करें।

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
