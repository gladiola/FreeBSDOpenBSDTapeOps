# FreeBSDOpenBSDTapeOps (Afrikaans)

Interaktiewe shell-skripte vir algemene magnetiese bandbewerkings met mt en tar.

## Vinnige gebruik

- Voer `./scriptedDemo.sh` op FreeBSD uit of `./scriptedDemo_openbsd.sh` op OpenBSD.
- Bevestig dat die band gelaai is en druk Enter by elke stap.
- Gebruik die skripte in `scripts/` vir A→B→C-argivering, oordrag, bandskryf, inventaris en herstel.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
