# FreeBSDOpenBSDTapeOps (Русский)

Интерактивные shell-скрипты для типовых операций с магнитной лентой через mt и tar.

## Быстрый запуск

- Запустите `./scriptedDemo.sh` на FreeBSD или `./scriptedDemo_openbsd.sh` на OpenBSD.
- Подтвердите, что лента загружена, и нажимайте Enter на каждом шаге.
- Используйте скрипты в `scripts/` для архивации A→B→C, передачи, записи на ленту, инвентаризации и восстановления.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
