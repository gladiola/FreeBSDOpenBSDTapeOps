# FreeBSDOpenBSDTapeOps (Українська)

Інтерактивні shell-скрипти для типових операцій із магнітною стрічкою за допомогою mt і tar.

## Швидке використання

- Запустіть `./scriptedDemo.sh` на FreeBSD або `./scriptedDemo_openbsd.sh` на OpenBSD.
- Підтвердьте, що стрічку завантажено, і натискайте Enter на кожному кроці.
- Використовуйте скрипти в `scripts/` для архівації A→B→C, передавання, запису на стрічку, інвентаризації та відновлення.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
