# FreeBSDOpenBSDTapeOps (Ελληνικά)

Διαδραστικά shell scripts για συνήθεις λειτουργίες μαγνητικής ταινίας με mt και tar.

## Γρήγορη χρήση

- Εκτελέστε `./scriptedDemo.sh` σε FreeBSD ή `./scriptedDemo_openbsd.sh` σε OpenBSD.
- Επιβεβαιώστε ότι η ταινία έχει φορτωθεί και πατήστε Enter σε κάθε βήμα.
- Χρησιμοποιήστε τα scripts στο `scripts/` για αρχειοθέτηση A→B→C, μεταφορά, εγγραφή σε ταινία, απογραφή και επαναφορά.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
