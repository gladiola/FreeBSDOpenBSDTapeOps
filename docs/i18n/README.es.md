# FreeBSDOpenBSDTapeOps (Español)

Scripts de shell interactivos para operaciones comunes de cinta magnética con mt y tar.

## Uso rápido

- Ejecute `./scriptedDemo.sh` en FreeBSD o `./scriptedDemo_openbsd.sh` en OpenBSD.
- Confirme que la cinta está cargada y siga cada mensaje presionando Enter.
- Use los scripts de `scripts/` para archivado A→B→C, transferencia, escritura en cinta, inventario y restauración.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
