# FreeBSDOpenBSDTapeOps (한국어)

mt와 tar를 사용한 일반적인 자기 테이프 작업용 대화형 셸 스크립트입니다.

## 빠른 사용법

- FreeBSD에서는 `./scriptedDemo.sh`, OpenBSD에서는 `./scriptedDemo_openbsd.sh`를 실행하세요.
- 테이프가 로드되었는지 확인하고 각 안내에서 Enter를 눌러 진행하세요.
- `scripts/`의 스크립트로 A→B→C 아카이브, 전송, 테이프 기록, 인벤토리, 복원을 수행하세요.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
