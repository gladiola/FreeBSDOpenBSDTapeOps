# FreeBSDOpenBSDTapeOps (Latina)

Scripta shell interactiva ad operationes communes taeniae magneticae utens mt et tar.

## Usus celer

- Exsequere `./scriptedDemo.sh` in FreeBSD aut `./scriptedDemo_openbsd.sh` in OpenBSD.
- Confirma taeniam oneratam esse et in singulis gradibus Enter preme.
- Utere scriptis in `scripts/` ad archivum A→B→C, translationem, scripturam in taeniam, inventarium, et restitutionem.

## Commands

```sh
./scriptedDemo.sh            # FreeBSD demo
./scriptedDemo_openbsd.sh    # OpenBSD demo
scripts/test-computer-a-b-c-integration.sh
```
